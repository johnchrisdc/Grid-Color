# Grid-Color
Minimalistic Watchface for Amazfit Pace

Based on the [GreatFit watchface](https://github.com/GreatApo/GreatFit) by [GreatApo](https://github.com/GreatApo)

## Widgets

**Screen On**
 - Battery percentage
 - Day of the week
 - Time
 - Day of the month
 - Month
 
**Screen Off**
 - Day of the week
 - Time
